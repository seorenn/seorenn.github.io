Seorenn Static Pages

어서오세요. Seorenn 의 특수한 페이지 저장소입니다. 이 곳은 그냥 제가 원할 때 마다 조금씩 글을 작성하여 즉석에서 업데이트 하는 용도로 만들어졌습니다. 따라서 미완성 글들이 넘쳐나며 동시에 업데이트도 자주 발생하는 그런 글들만 모여 있습니다.

주로 IT나 프로그래밍, 개발과 관련된 정보 위주로 구성되어 있습니다만 간혹 이상한 도구에 관한 메모를 쓸 때도 있습니다. 사실상 개인 메모장 대용입니다. 😃

모든 글은 Emacs 상에서 Org Mode로 글을 작성해서 Org Publish를 이용해 퍼블리시 합니다.

탐색 시작하기 좋은 페이지

몇 가지 분류로 정리한 가이드 페이지들입니다.

프로필(Profile)

  • Profile: 사적인 정보(?)는 없는 프로필입니다.

에디터(Editors)

  • Emacs: 이맥스와 관련된 글들을 모아 놨습니다.
  • Doom Emacs: 이맥스와 관련된 글들 중 둠 이맥스와 관련된 것들만 따로 모아 봤습니다.
  • Spacemacs: 스페이스맥스를 간략히 정리하는 글입니다.
  • Vim: 빔 이데터에 관한 메모를 모으는 페이지입니다.

프로그래밍 언어(Programming Language)

  • Swift: 가장 좋아하는 프로그래밍 언어인 스위프트와 관련된 글들을 모은 페이지입니다.
  • Python: 두 번째로 좋아하는 프로그래밍 언어인 파이썬과 관련된 글들을 모은 페이지입니다.
  • Lisp: 싫어하지만(?) 억지로 써야 하는 리습과 관련된 글들을 모아둔 페이지입니다.
  • Javascript: 자바스크립트와 관련된 글들을 쓸 예정입니다.
  • CSS(Cascading Style Sheets): CSS에 관한 몇몇 정보를 모으는 페이지입니다.
  • Bourne Shell(sh): 셸 스크립팅에 관한 내용이 주인 sh 페이지입니다.

개발용 도구들

  • Git: 깃과 관련된 정보를 모아둔 페이지입니다.
  • Docker: 도커와 관련된 정보를 모아둔 페이지입니다.
  • kubectl: 쿠버네티스 CLI 도구인 kubectl 사용법을 모아둔 페이지입니다.

기타 도구들

  • Tmux: Tmux 사용법을 정리하는 페이지입니다.
  • Homebrew: 맥 용 패키지 관리자 흠브루에 관한 내용을 정리하는 페이지입니다.
  • iconv: 인코딩 변환 도구인 iconv의 사용법을 간략히 정리합니다.
  • find: 파일 검색 도구인 find 유틸리티의 사용법을 정리하는 페이지입니다.
  • caffeinate: 맥 공식 잠자기 방해 도구인 카페이네이트 사용법을 간략히 정리하는 페이지입니다.

분류하기 애매한 것들

  • MongoDB: 몽고DB와 관련된 내용을 정리하는 페이지입니다.
  • Github Actions: 깃헙 액션에 관한 내용을 정리하는 페이지입니다.
