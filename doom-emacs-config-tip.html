폰트 설정

폰트나 텍스트 크기를 설정하려면 config.el 에 아래와 같은 식으로 코드를 넣을 수 있습니다. 

(setq doom-font (font-spec :family "Menlo" :size 14))

이 설정은 고정폭 폰트에만 유효합니다.

줄 간격 설정

줄 간격(line spacing 혹은 line height) 설정은 아래와 같은 식의 코드를 config.el 에 넣으면 설정할 수 있습니다. 

(setq-default line-spacing 2)

창 초기 위치 및 크기 설정

예를 들어 위에서 120픽셀, 좌측에서 50픽셀 떨어진 곳에 80x24 크기로 Emacs 창을 띄우고 싶다면 아래 코드를 config.el 에 넣으면 됩니다. 

(when window-system
  (setq initial-frame-alist
        '((top . 120)
          (left . 50)
          (width . 80)
          (height . 24))))

참고로 top, left 값은 픽셀인 반면, width와 height 값은 문자 갯수로 표현합니다. 즉 width 80 height 24 값은 가로 80컬럼 세로 24로우의 의미입니다.

배경색이 버퍼마다 달라요?!

Doom Emacs에 포함된 solaire-mode는 배경색을 임의로 변경하는 확장입니다. 이걸 비활성화 하면 배경색이 바뀌는 문제는 해결됩니다.

SPC f p 로 packages.el 파일을 열어서 아래 한 줄을 넣어줍시다. 

(package! solaire-mode :disable t)

수정 후에는 ~/.emacs.d/bin/doom refresh -p 커맨드를 잊지 맙시다.

이모티콘 입력기

이모티콘(emoji) 입력은 emojify 라는 확장을 설치해서 할 수 있습니다. packages.el 에 아래 내용을 넣어줍시다. 

(package! emojify)

그리고 아래 코드를 config.el 에 넣어줍시다. 

(add-hook! 'after-init-hook #'global-emojify-mode)

저장 후 터미널에서 아래 커맨드를 입력해서 패키지가 설치될 수 있도록 합니다. 

~/.emacs.d/bin/doom refresh

이제 Emacs를 재시작하고 SPC : 를 눌러 emojify-insert-emoji 함수를 찾아봅시다. 이 함수를 실행시키면 이모티콘을 검색할 수 있는 다이얼로그가 뜹니다.

프로젝트 추가하기

Git 저장소 등 몇몇 디렉토리는 방문 시 자동으로 Projectile로 프로젝트 디렉토리로 인식되고 자동으로 등록됩니다. 하지만 이런 특수한 디렉토리 말고 일반 디렉토리의 경우는 프로젝트로 인식되지 않습니다.

이런 일반 디렉토리를 프로젝트로 등록하려면 아래 코드를 config.el 에 넣어줍시다. 

(projectile-add-known-project "~/boo/bar/project")

물론 갯수는 상관 없습니다. 원하는 만큼 등록해봅시다.

컬러 테마 사용자화

컬러 테마를 커스터마이징 하기 위해서 doom-themes-set-faces 함수가 준비되어 있습니다. 

(doom-themes-set-faces nil
  '(foo-face :foreground #123456)
  '(bar-face :background nil))

첫 매개변수 nil 은 테마 이름인데 nil 로 설정할 경우 모든 테마에 적용되게 됩니다.

페이스(색상이나 폰트 등의 정보)를 찾을 때는 커서를 원하는 위치에 놓고 SPC h F 를 눌러보면 쉽게 알 수 있습니다.
