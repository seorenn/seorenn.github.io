Emacs 설치

Doom Emacs 는 Emacs 설정 스크립트이기 때문에 Emacs 는 별도로 설치해야 합니다. Emacs 설치 글을 참고하여 설치를 합니다. 이미 설치가 되어있다면 다음으로 넘어갑니다.

Doom Emacs 설치

우선 Doom Emacs 저장소를 clone 해야 합니다. git 이 설치되어 있다는 가정하에 아래와 같은 방식으로 클론할 수 있습니다. 

git clone https://github.com/hlissner/doom-emacs ~/.emacs.d

클론이 완료가 되면 일단 클론한 디렉토리로 진입해 봅시다. 

cd ~/.emacs.d

이제 한가지 결정을 해야 합니다. 안정 버전을 사용할 것인가 아니면 개발 버전을 사용할 것인가 말이지요.

Doom Emacs 도 Spacemacs 와 비슷하게 안정 버전 릴리즈가 굉장히 느린 편입니다. 따라서 신기능을 최대한 빠르게 써보고 싶다면 개발 버전을 사용하는 것을 추천합니다. 아래의 커맨드로 개발 버전으로 바꿀 수 있습니다. 

git checkout develop

안정버전을 사용하겠다면 위의 커맨드는 생략해도 무방합니다.

이제 필요한 기본 설정 파일 및 패키지들을 설치해야 합니다. .emacs.d 디렉토리에 있는 상태로 아래 커맨드를 입력하면 됩니다. 

bin/doom install

이렇게 하면 몇 가지 필요한 옵션을 물어오고 이에 대답을 끝내면 각종 패키지 다운로드가 진행됩니다. 조금은 오래 걸릴 수도 있으니 잠깐 커피 타임을 가지면 좋을 것 같습니다.

설치가 완료되면 이제 기본적인 Doom Emacs 가 셋업된 상태입니다. Emacs 를 기동시켜서 제대로 동작하는지 확인해 봅시다.

추가 설치

Doom Emacs 가 제대로 기동 되었다면 최초 한 번은 SPC : 를 누른 후 all-the-icons-install-fonts 를 입력해 줍시다. 필요한 폰트 파일을 다운로드 받아서 알아서 설치해 줄 것입니다.

이 외에 미비한 사항은 doctor 커맨드를 통해 파악해 봅시다. 

bin/doom doctor

업그레이드

Doom Emacs 도 개발이 잦아서 변화가 자주 있는 편입니다. 또한 각종 확장들도 별도로 업그레이드가 이뤄집니다. 따라서 업그레이드 하는 방법을 꼭 알아두는 편이 좋습니다.

터미널에서 아래의 명령으로 Doom Emacs 및 확장을 한 번에 업그레이드 할 수 있습니다. 

~/.emacs.d/bin/doom upgrade

혹은 upgrade 의 축약어인 up 을 이용할 수도 있습니다. 

~/.emacs.d/bin/doom up

이렇게 하면 Doom Emacs 자체 및 관련된 패키지들이 몽땅 업그레이드가 됩니다.

주의: 혹여 실수로 git pull 같은 명령은 실행하지 않는 편이 좋습니다. 잘못하다 망가지면 되돌리는 것이 귀찮아질 수도 있습니다.
