SVG는 Scalable Vector Graphics의 약자이자 웹 브라우저에서 사용되는 벡터 이미지 포맷이다. 비슷한 포맷으로 Adobe Illuatrator(.ai) 포맷이 있지만 SVG는 XML로 표현된 오픈 표준의 포맷이라는 큰 장점이 있어서 상대적으로 많이 사용된다.

벡터이기 때문에 면과 면을 채우는 색, 그리고 테두리 색 등으로 이미지를 구성한다.

팁

macOS에서 SVG 파일을 PNG로 바꾸기

srvg-convert 라는 CLI 유틸리티로 변환이 가능하다. 

rsvg-convert -h 512 original.svg > converted.png

-h 512 옵션은 높이(height)를 512 픽셀로 해서 비율을 설정하는 옵션이다. SVG 자체는 해상도가 없기 때문에 비트맵으로 만들려면 반드시 크기가 있어야 하기 때문에 원하는 크기를 설정해 주자.

만약 높이가 아닌 폭(width)을 정하고 싶다면 -h 옵션 대신 -w 옵션을 사용해보자.

참고로 rsvg-convert 유틸리티는 librsvg 패키지에 들어있다. 없다면 Homebrew로 설치해주자. 

brew install librsvg
