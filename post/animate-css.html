Animate.css

2020년 6월 11일 작성

Animate.css는 크로스 브라우저용 애니메이션 CSS 라이브러리다. 클래스 지정 만으로 간단한 애니메이션을 구현할 수 있다. 뭐 말로 설명하는 것 보다는 직접 보는 게 좋을 것 같다.

🌏Animate.css

우측의 사이드바에서 다수의 애니메이션 스타일을 누르면 화면 가운데서 바로 효과가 나타나기에 어떤 애니메이션이 지원되는지 파악하기 쉬운 것 같다.

설치도 npm이나 yarn으로 간단히 되고 사용도 간단해 보이기는 한데, 문제는 이걸 어디다 써먹어야 할지 잘 모르겠다는 점이다. 개인적으로는 가벼운 웹페이지를 선호하는 터라 오히려 불필요한 곳에서 이런 애니메이션을 쓰면 반감만 더해질 지도 모르겠다.

하지만 눈에 띄어야 할 부분을 강조할 때 쓴다면 그야말로 용도에 맞는 사용일 것 같다. 일단 메모해 두고 나중에 필요해지면 그 때 써먹어 봐야겠다.
