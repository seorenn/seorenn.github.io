  • 사용법

afplay는 Audio File Play의 약자로 macOS용 CLI 오디오 파일 플레이어 유틸리티다.

사용법

afplay /foo/bar/audio.mp3

포그라운드로 돌기 때문에 ^C 로 플레이 도중 종료 시킬 수 있다.

-h 옵션을 통해 도움말을 볼 수 있는데 몇 가지 편의 기능이 있다.

  • -t N 혹은 --time N 옵션으로 N 초 만큼만 플레이 하게 할 수 있다.
  • -v N 혹은 --volume N 으로 N 만큼 볼륨 조정. 단 백분율이 아님에 주의. 1이 현재 볼륨이 아닐까 유추 중.
  • -r N 혹은 --rate N 옵션으로 플레이 속도 조절. 1이 원래 속도로 유추됨.
