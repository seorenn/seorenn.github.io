flake8

flake8 은 Python 코드를 점검해주는 도구이자 파이썬 패키지입니다. 일부 오류가 있을 법한 코드나 미사용 import 등등 여러 문제점을 알려주면서 PEP8 코딩 컨벤션에 맞지 않는 것들을 알려줍니다.

설치법

파이썬 패키지이기 때문에 pip로 그냥 설치하면 됩니다. 

pip install flake8

사용법

CLI로 그냥 실행시키면 됩니다. 

flake8 /foo/bar/some/file.py

이렇게 하면 체크된 내용을 텍스트로 알려줍니다.

근데 요즘은 이렇게 단독으로 실행시키기 보다는 에디터나 IDE 등에서 자동으로 호출해서 체크하는 방식으로 동작하는 경우도 많기 때문에 굳이 손수 실행시킬 일은 거의 없는 것 같습니다.

설정 파일

전역 설정파일은 ~/.config/flake8 파일에, 프로젝트 단위 설정은 프로젝트 루트에 .flake8 혹은 setup.cfg 파일로 설정 내용을 작성하면 됩니다.

대충 아래와 같은 식으로 작성합니다. 

[flake8]
max-line-length = 80
ignore = F123, E456
exclude = file_to_exclude.py

max-line-length 는 컬럼 사이즈입니다. ignore 는 체크하지 않을 에러 코드를 작성하고, exclude 는 체크하지 않을 파일을 작성할 수 있습니다. 물론 설정 내역은 이 보다 더 광범위할 수 있으니 공식 매뉴얼을 참조합시다.

파일 별로 일부 경고 무시하기

커맨드 라인 옵션으로 특정 체크를 무시할 수 있습니다. 

flake8 --ignore F123,E456 FILEPATH

근데 이렇게는 별로 안 쓰일테니 일단 넘어갑시다.

소스 코드에서 특정 라인 하나만 무시하려면 라인 끝에 아래와 같은 주석으로 표기가 가능합니다. 

pass  # noqa: F123,E456

코드를 표기하지 않고 noqa 만 명시하면 모든 오류 체크가 무시됩니다.

위와 비슷하게 아래처럼 아예 해당 파일의 체크를 하지 않게 할 수도 있습니다. 

# flake8: noqa

이런 식으로 쓸 일은 별로 없겠지만요.
