CSS를 HTML문서에 사용하기

HTML에 스타일을 등록하는 방법은 대략 3가지가 있습니다.

첫 째로 바로 태그에 스타일을 입히는 방법입니다. 

<p style="font-size: 2rem">...</p>

간단하게 한 태그의 스타일만 바꿀 때는 편하지만 여러 곳에서 같이 사용할 스타일을 일일이 등록하는 것은 여러 면에서 피곤한 일입니다. 따라서 이런 방식은 특정 태그 하나만 별도로 스타일을 수정하는 경우를 제외하곤 그다지 추천되지는 않습니다.

다른 방법으로 HTML 코드에 스타일을 블럭으로 추가할 수 있습니다. 일반적으로 head 태그 안에 아래와 같은 코드를 넣습니다. 

<style>
p {
  font-size: 2rem;
}

...
</style>

태그 하나에 스타일을 입히는 것 보다는 여러 면에서 유용한 방법이지만 이 역시 하나의 HTML 파일에 일일이 기록해야 한다는 점에서 비효율적으로 여겨집니다. 이런 형식은 특정한 한 파일에만 필요한 스타일을 기록하는 용도로써는 추천됩니다.

마지막으로 가장 많이 사용되는, 별도의 스타일 파일을 작성하는 식입니다. 아래와 같은 스타일을 style.css 라는 파일로 작성했다고 칩시다. 

p {
  font-size: 2rem;
}

...

이 파일을 HTML 파일에 적용하기 위해서는 아래와 같은 코드를 해당 HTML 파일에 넣어주면 됩니다. 

<link res="stylesheet" href="style.css" />

파일 경로는 상황에 맞게 입력하는 것이니 넘어가고, 하여간 이런 식으로 스타일 파일을 별도로 작성해 두면 모든 HTML 파일에 일일이 적용하는 것 보다는 훨신 편리합니다. 단점은 저 코드가 잘 기억나지 않는다는 점입니다.😓

단어를 자르지 않기

문장이 길어져서 다음 줄로 넘어갈 때 단어를 중간에 자르면 가독성에 문제가 있을 수 있습니다. 이럴 때는 아래와 같은 스타일을 넣을 수 있습니다. 

word-break: keep-all;

물론 호불호는 있을 수 있습니다.

둥근 모서리

모서리를 둥글게 하려면 아래처럼 할 수 있습니다. 

border-radius: 8px;

저기엔 숨겨진 많은 옵션이 있습니다.

div 블럭 가운데 정렬

큰 div 블럭 안에 가로 640 픽셀의 작은 div 블럭을 가운데로 정렬하려면 아래처럼 할 수 있습니다. 

.some_block {
  max-width: 640px;
  margin: 0 auto;
}
마진을 쓰기 때문에 테두리는 제대로 쓰질 못 하겠네요. 그럴때는 그냥 이 안에다 다른 div 블럭을 만들면 되긴 하지요.

소수점 자리로 정렬

테이블 틍에서 실수를 표시할 때 소수점 기준으로 표시하려면 아래와 같은 스타일을 추가할 수 있습니다. 

.table {
  font-variant-numeric: tabular-nums;
}

달러나 유로 단위의 경우 소수점 아래도 표기하는데 이럴 때 유용할 것 같습니다.
