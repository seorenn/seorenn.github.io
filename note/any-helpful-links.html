생활

  • 🌏쓰레기 백과사전: 쓰레기 분리수거 방법이 정리되어 있다.

비디오 다운로더

  • 🌏9xbuddy: 웹 인터페이스 비디오 다운로더. 웹 UI에서 주소를 붙여넣거나 혹은 https://9xbuddy.com/https://www.youtube.com/watch?=foobar 같은 주소 직접 입력 방식도 지원한다.

시간 때우기

  • 🌏Drive & Listen: 세계 각지의 수도를 차를 타고 달리는 듯한 화면을 계속 보여주며 해당 국가 언어의 라디오 같은 것을 계속 들려주는 사이트. 무분별(?)하게 시간을 때우기 좋은 것 같다. 멍 때리기에도 좋은 듯 하다.
  • 🌏웹 기반 대항해시대2: 아마도 웹에서 돌아가는 DOSBox 기반인 것 같다. 정말 옛날에 나왔던 그 명작 게임이다.

맥, PC 제어 관련

  • 🌏Barrier (GitHub): 키보드나 마우스를 공유하기 위한 도구. 맥이나 PC등 동종 혹은 아종 하드웨어가 여럿 있을 때 하나의 키보드와 마우스로 모두를 제어하는 데 쓸 수 있다는 것 같다. Wifi로 연결되기 때문에 레이턴시나 끊김은 좀 있다는 것 같다.

소셜 미디어 관련

  • 🌏Always Up-to-Date Guide to Social Media Image Sizes: 각종 SNS 등등의 소셜 미디어 서비스에서 각 항목 별로 추천되는 이미지 사이즈가 정리되어 있다.

분류하기 힘든 링크들

  • 🌏ㅅㅇ) 현실적인 학교폭력 대처법.jpg: 합법적인지 효과는 있는지 검증되었다고는 보기는 힘들지만 법적으로 개인이 학교에 대응하기 위한 방법을 상싱적으로 정리해 둔 것 같다. 사이트가 일단 유머 사이트라는 것에 유의하자.
  • 🌏현재 이슈터진 KT인터넷 속도 해결방법: 혹시나 KT 인터넷을 쓰는데 자신이 쓰는 회선보다 속도가 느린것 같다면 시험해 볼 가치는 있을 것 같다. 간략히 정리하자면 🌏access.olleh.com 에 접속해서 사용중인 부가서비스 항목의 볼드표시된 텍스트를 클릭해서 뜨는 인증 폼에서 아이디 reset 비밀번호 reset1 을 입력해보자.
