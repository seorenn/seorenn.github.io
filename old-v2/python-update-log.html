Python 의 버전 별 업데이트 내역을 간략히 정리합니다. 내부적인 변화는 특별한 것이 아니면 생략합니다. 다만 3.8 버전 부터 정리하고 이후 업데이트를 보장하지 않습니다.

3.8

  • := (Walrus) 오페레이터: 좁은(?) 지역 변수. Swift 의 if let 과 용도가 비슷합니다. 조건문 뿐만 아니라 List Comprehension 등에서도 활용이 가능합니다.
  • f-string 추가 표현식: f'name={value}' 대신 f'{name}'= 문법을 사용할 수 있습니다. 관련된 내용은 Python 3 F-string 글을 참고합시다.
  • / 오퍼레이터: 함수 매개변수 리스트 구분자. f(a, b, /, c, d, *, e, f) 매개변수를 나열할 때 / 앞쪽은 위치로만 표기 가능하며, / 와 * 사이는 위치와 키워드 관계 없이 사용 가능, * 이후는 키워드 표기법만 사용 가능합니다.
