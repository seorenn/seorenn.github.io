vscode

vscode 는 Microsoft Visual Studio Code의 약자로 보통 부르는 이름이다. 즉 마이크로소프트에서 출시한 코드 에디터다. 웹 기반 엘렉트론(Electron)으로 구현된 GitHub의 Atom 에디터를 기반으로 개발된 것으로 알려져 있다.

거의 대부분 분야의 개발용 IDE로 만들어 주는 엄청난 플러그인 생태계가 구축되어서 현재는 Vim이나 Emacs를 압도적으로 뛰어넘는 사용자 층을 가지고 있다. 사실상 에디터계의 독점 소프트웨어가 되었다.

설치

macOS에서는 역시 Homebrew로 설치하는 편이 쉽고 간편하다.

brew install --cask visual-studio-code

물론 🌏공식사이트에서 직접 다운로드 받아서 설치해도 된다.

터미널에서 띄우기

code 라는 실행 바이너리가 제공된다. 이 명령으로 몇 가지 제어가 가능하다.

아래 명령은 현재 디렉터리를 vscode의 새 창으로 연다.

code .

위에서 . 은 현재 디렉터리를 뜻하는 그 기호가 맞다. 즉 다른 경로를 입력하면 해당 디렉터리나 파일을 바로 열 수 있다.

만약 현재 열려있는 vscode 창에다 디렉터리를 열려면 -r 옵션을 쓸 수 있다.

code -r .
