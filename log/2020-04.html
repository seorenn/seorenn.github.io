2020년 4월 29일

🌏Prometheus + Grafana로 모니터링하기 (1): 프로메테우스는 로그 수집기고 그라파나는 시각화 도구로 이해하면 될까? 당장은 뭐에 써야 할지 감이 오질 않는다. 공공(public) 로그를 제공할 이유도 없고 말이지. 그래도 알아두면 언젠가는 써 먹을 날이 오지 않을까?

🌏Package Manager Binary Dependencies: SwiftPM에 바이너리 의존성이 들어간다는 것은 private framework의 배포도 가능하게 된다는 것일까? 하긴 소스 기반이 아닌 다른 사업 모델을 가진 기업에서 배포하는 프레임워크로 얼마든지 있을 수 있다. 좋은 변화인 건 확실한 듯.

🌏맥북프로가 뜨겁거나 kernel task 의 CPU 사용량이 높다면 왼쪽말고 오른쪽으로 충전하세요: 뭔 개소리인가 했는데 이내 납득되었다. 열이 주로 발생하는 곳 근처의 포트를 사용하면 당연히 그 쪽이 더 과열될 수 밖에 없고 결국 CPU 스로틀링이 걸릴 수 밖에 없다. 이런 머리 좋은 사람들.

🌏Python 2.7.18, the last release of Python 2: Python 2 버전의 공식적인 가장 마지막 버전이 릴리즈 되었다. 하지만 이미 사망 선고를 받고 무덤에 묻힌 상태라 영 감흥이 없다.
