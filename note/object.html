컴퓨터 프로그래밍 개념 1

오브젝트(Object, 객체)는 OOP(Object Orient Programming, 객체 지향 프로그래밍)에서 클래스(Class)를 실체화 시킨 인스턴스를 특별히 부르는 이름이다.

사실상 인스턴스와 동일하지만 OOP에 한정된 약간은 좁은 범위의 인스턴스다.

컴퓨터 프로그래밍 개념 2

C나 C++ 구현 파일을 컴파일해서 나온 이진(Binary) 파일을 오브젝트 파일이라고 부른다. 실제로 확장자가 .o 이다. 이 파일을 링커로 합치면 일반적인 라이브러리 파일이나 실행 파일이 된다.
