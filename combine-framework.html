이게 뭘까?

Combine 은 애플의 반응형 프로그래밍 라이브러리입니다. 어 근데 반응형이요? 이 '반응형' 이라는 말이 참 괴롭습니다. 이름만 봐서는 도저히 이해가 안 되니깐요.

우선 코드부터 시작합시다. 

import Combine

네. Combine 은 프레임워크입니다. 그래서 임포트해야 쓸 수 있습니다.

그런데 뭐가 무엇에 반응하는 건데요?

다짜고짜 예제

일단 아래 예제를 살펴봅시다. 

let _ = Just(10)
    .map { value -> Int in
        return value * 2
    }
    .sink { value in
        print("Finished with value \(value)")
    }

이 코드가 실행되면 어이없게도(?) Finished with value 20 이라는 결과가 찍히고 종료됩니다. 마치 map 안의 내용과 sink 의 내용이 순차적으로 실행된 것 처럼요.
