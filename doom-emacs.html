Doom Emacs

Seorenn Static Pages

Doom Emacs 는 Emacs용 Community Driven Configuration의 양대산맥인 Spacemacs와 함께 유명한 이맥스용 설정 프로젝트입니다. 직접 과도하게 세팅해서 사용하는 것이 귀찮은 저와 같은 유저라면 이런 설정도 사용해보면 괜찮을 것 같습니다.

Doom 도 Spacemacs 와 비슷하게 Vim 키맵을 구축시켜 주는 evil이라는 확장을 토대로 Vim 키맵에 기반한 Emacs 환경을 셋업하고 이와 어울리는 단축키 환경을 만들어 놓은 것이 특징입니다. 아마도 스페이스맥스에서 뻗어나온 뿌리에서 새로 자란 거대하면서도 다른 나무 같은 녀석이라고 볼 수 있겠네요.
