파이썬(Python)은 네덜란드 출신의 귀도 반 로썸(Guide van Rossum)에 의해 탄생한 프로그래밍 언어다. 동적 타이핑을 사용하는 인터프리터 언어라는 점도 특징이지만, 역시 들여쓰기를 블록 구분으로 사용하는 것을 문법으로 강제하는 언어라는 점이 가장 큰 특징이다.

설치

macOS 의 경우 Homebrew 를 통해 설치하거나 혹은 pyenv 로 쉽게 설치할 수 있다.

다만 최근 Homebrew 를 통해 설치하는 경우 3.x 버전이 설치되며 python3 라는 이름으로만 실행시킬 수 있게 설치된다. 즉 python 커맨드로 실행되는 버전은 시스템의 기본 파이썬 바이너리다. 만약 python 커맨드로 새로 설치한 파이썬 3를 실행시키고자 한다면 아래와 같은 경로 설정을 추가해야 한다. 

export PATH="/usr/local/opt/python/libexec/bin:$PATH"

Bytes 관련 팁

Python 3 부터 Bytes를 표시하는 방법이 바뀌었다. 

b'some string'

b 접두어를 붙인 문자열은 Binary String 혹은 binstring 이라 부르는 타입이다.

바이트 배열과의 관계는 이렇다. 

>>> b = b'binstring'
>>> ba = bytearray(b)
>>> bb = bytes(ba)
>>> b == bb
True

기존 2.x 에서 쓰던 바이트 버퍼와 관련된 코드들이 아마도 이런 변경 때문에 오류가 나는 경우를 겪을 수 있을 것이다.

파일 혹은 디렉토리(폴더) 관련 팁

경로에서 파일명만 얻기

>>> import os
>>> os.path.basename('/foo/bar/test.txt')
'test.txt'

파일명에서 확장자 분리하기

>>> import os
>>> os.path.splitext('test.txt')
('test', '.txt')

splitext 는 파일명 뿐만이 아니라 경로(path)에서도 동작한다. 예를 들어 /foo/bar/text.txt 를 입력할 경우 /foo/bar/text 와 .txt 로 분리됩니다.

파일 혹은 디렉토리 구분하기

>>> import os
>>> os.path.isfile('/foo/bar/some/file.txt')
True
>>> os.path.isdir('/foo/bar/some/file.txt')
False

디렉토리 안의 파일 목록 얻기

>>> import os
>>> os.listdir('.')
['foo.a', 'bar.b', 'somedir', ...]

사전(Dictionary)형 관련 팁

합치기

일반적인 방법으로 update 메서드를 활용할 수 있다. 

>>> a = {'a':1, 'aa': 2}
>>> b = {'b':3, 'bb': 4}
>>> a.update(b)
>>> a
{'a': 1, 'aa': 2, 'b': 3, 'bb': 4}

파괴적인(destructive)한 방법이라 약간은 꺼려지기도 한다.

이 외의 트릭으로 아래 방법이 있다. 

>>> a = {'a': 1, 'aa': 2}
>>> b = {'b': 3, 'bb': 4}
>>> {**a, **b}
{'a': 1, 'aa': 2, 'b': 3, 'bb': 4}

Immutable 한 스타일로 쓸 수가 있어서 개인적으로 더 마음에 든다.
