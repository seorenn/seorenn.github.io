Finder (파인더)

New Folder With Selection

macOS Ventura 부터 New Folder With Selection(선택 항목을 포함하는 새로운 폴더) 기능이 생겼다. 이름처럼 특정 디렉터리를 생성해서 여기에 선택된 파일을 옮기는 기능이다. 사용 방법도 간단한데 파인더에서 원하는 파일을 선택하고 우클릭을 눌러서 나오는 메뉴 최상단에서 찾을 수 있다.

Reminder (미리알림)

순서 바꾸기 (reorder)

미리알림 목록의 순서를 마음대로 바꾸려먼 체크박스 좌측의 빈 여백을 왼클릭으로 드래그하면 된다. 괜히 우클리나 여기저기 드래그 등은 안 되니 알아두자. 참고로 macOS Ventura 기준이다.
