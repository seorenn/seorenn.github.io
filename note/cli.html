CLI

2021년 5월 2일 수정

CLI는 Command Line Interface의 약자로 직역하여 명령 줄 인터페이스다. 즉 컴퓨터가 수행할 명령을 키보드를 이용해 직접 입력하는 방식의 인터페이스다. 보통 터미널을 통해 직접 커맨드를 입력하는 방식으로 이해할 수 있다.

비슷한 용어로 CUI (Console User Interface)가 있고 거의 상반되는 의미로 GUI (Graphical User Interface)가 있다.

일반적으로 사용자 명령어를 셸을 통해 분석해서 시스템에 전달한다. 따라서 CLI에는 셸이 크게 관계하고 있다고 볼 수 있다.

GUI에 비해 불편하다고 느낄 수 있으나 CLI 특유의 장점 또한 많기 때문에 아직도 많이 쓰이고 있는 방식의 사용자 인터페이스다.
