단어를 자르지 않기

문장이 길어져서 다음 줄로 넘어갈 때 단어를 중간에 자르면 가독성에 문제가 있을 수 있습니다. 이럴 때는 아래와 같은 스타일을 넣을 수 있습니다. 

word-break: keep-all;

물론 호불호는 있을 수 있습니다.

둥근 모서리

모서리를 둥글게 하려면 아래처럼 할 수 있습니다. 

border-radius: 8px;

저기엔 숨겨진 많은 옵션이 있습니다.

div 블럭 가운데 정렬

큰 div 블럭 안에 가로 640 픽셀의 작은 div 블럭을 가운데로 정렬하려면 아래처럼 할 수 있습니다. 

.some_block {
  max-width: 640px;
  margin: 0 auto;
}
마진을 쓰기 때문에 테두리는 제대로 쓰질 못 하겠네요. 그럴때는 그냥 이 안에다 다른 div 블럭을 만들면 되긴 하지요.

소수점 자리로 정렬

테이블 틍에서 실수를 표시할 때 소수점 기준으로 표시하려면 아래와 같은 스타일을 추가할 수 있습니다. 

.table {
  font-variant-numeric: tabular-nums;
}

달러나 유로 단위의 경우 소수점 아래도 표기하는데 이럴 때 유용할 것 같습니다.
