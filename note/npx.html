npx

npx는 npm 5.2.0 혹은 그 이후의 버전 설치 시 사용할 수 있는 CLI 도구로 원격에 등록된 npm 패키지의 바이너리 혹은 스크립트를 실행시키는 용도로 사용한다. 그래서 npm으로 패키지를 설치하지 않고도 간단한 작업을 대신 수행할 수 있다.

사용 예

대표적으로 Nuxt.js 프로젝트를 시작하기 위해서 아래 커맨드를 사용할 수 있다.

npx create-nuxt-app <project-name>

혹은 경쟁 상대(?)인 React 프로젝트를 시작할 때도 사용할 수 있다.

npx create-react-app <project-name>
