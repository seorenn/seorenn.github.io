이맥스(Emacs)는 커스터마이징이 가능한 유명한 에디터 중 하나입니다. 이름의 유래는 Editor MACroS 의 약자로 에디터 매크로 모음이라고도 볼 수 있는데 실제로 이맥스 자체는 C 로 구현된 Emacs Lisp 엔진과 이를 이용해 만들어진 수 많은 Emacs Lisp Function 으로 구성되어 있습니다.

How to debug freezing?

모종의 사유로 인해 Emacs 가 멈쳤을 경우 SIGUSR2 신호를 주면 백트레이스를 얻을 수 있다고 합니다.

우선 실행 중인 Emacs 의 PID 를 알아야 합니다. 

ps -ef | grep Emacs | grep -v grep

결과에서 두 번째 컬럼의 숫자가 PID 입니다.

그 다음 이맥스를 멈추게 만듭니다. (…)

이제 Emacs 에 SIGUSR2 를 던져봅시다. 

kill -USR2 PID

위에서 PID는 앞서 구한 Emacs 의 PID 숫자를 넣어야 합니다.
