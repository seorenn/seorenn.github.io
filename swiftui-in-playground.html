OS 공통

iOS든 macOS든 가리지 않고 아래 코드로 플레이그라운드에서 SwiftUI를 시험해 볼 수 있습니다. 

import SwiftUI
import PlaygroundSupport

struct ContentView: View {
    var body: some View {
        Text("Hello Playground!")
    }
}

PlaygroundPage.current.setLiveView(ContentView())

실행시킨 후 창 우측 상단에 있는 'Adjuest Editor Options' 메뉴에서 Live View 를 켜주면 뷰의 모습을 볼 수 있습니다.

macOS 에서는 텍스트가 상단에 표시되고, iOS의 경우는 텍스트가 가운데에 표시된다는 차이가 있습니다.

아직 SwiftUI를 플레이그라운드에서 완벽히 돌리기엔 무리가 있다는 점을 알아둡시다.

참고로 하단의 코드를 PlaygroundPage.current.liveView = ContentView() 처럼 쓰고 싶을 수도 있는데 이렇게 하면 PlaygroundLiveViewable 을 따르지 않는다며 오류가 납니다. Objective-C 구현체가 아니기 때문에 setLiveView(...) 와 liveView = ... 코드는 다른 방식으로 동작한다고 생각합시다.
