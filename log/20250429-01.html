메인 블로그 티스토리로 이전

기타 // 2025년 04월 29일 작성

한동안 GitHub Pages를 통해 개인 블로그를 운영하고 있었지만 한동안 고민에 휩싸였었다. 꾸준히 글을 올리고 있었지만 왜인지 검색 트래픽이 계속 감소하고 있어서다. 이유를 찾고는 있지만 Bing이 사이트를 차단한 사소한(?) 사건을 제외하면 특별한 이유를 모르겠다.

그래서 약간의 실험을 다시 해보기로 했다. 메인 블로그를 티스토리📎로 다시 이전하는 것이다. 티스토리는 자체광고도 불만족스럽고 최근 업데이트도 전혀 없고 재정안정성이 불안정한 것처럼 보여서 탈출을 결심했었지만 그래도 여러 면에서 장점은 여전히 가지고 있기도 하니 말이다.

GitHub Pages가 답답한 가장 큰 이유는 사진을 많이 첨부하기에 부담스럽다는 점이다. 1GB라는 저장소 공간 제한은 생각보다 크다. 사진을 안 올리면 꽤 많은 양의 글을 쓸 수 있지만 한 페이지에 사진을 두 장만 올려서 금방 한계에 봉착하게 된다. 하지만 사진 없이 글만 올리면 뭔가 이해도를 떨어뜨리는 요소가 되기도 하는 듯해서 상당한 고민이 있었던 사유다.

모든 것을 관리하는 것도 부담스럽다. 최근 Bing이 이 사이트를 차단했는데 이유를 알 수가 없는 게 너무 찝찝하다. 다만 티스토리는 Bing에게 RSS와 사이트맵을 차단시킨 막장이 있기는 해서 도긴개긴일 지도 모르겠다.

Emacs로 글을 쓰고 관리하는 것이 장단점은 분명히 있다. 글 쓰기가 편하고 하기 나름이지만 링크를 구성하거나 하는 것에서 유연성이 굉장히 크다. 하지만 외부에 나가게 되면 업데이트가 거의 불가능해지는 단점은 극복하기가 어려웠다.

물론 언제 어떻게 망할지도 모를 티스토리라 대비는 하고 있어야 할 것 같다. 아마도 티스토리가 망하면 정적 블로그를 GitHub Pages가 아닌 Cloudflare로 이전하여 운영할 가능성이 높을 것이다. 거긴 용량이나 트래픽 제한이 없으니 말이다.

어쨌든 그리하여 당분간 티스토리가 메인 블로그가 된다. 하지만 여전히 이 블로그도 남아있을 것이며 사소한 업데이트는 계속될 것이다. 그리고 관련된 글은 링크를 통해 쌍방으로 연결될 것이다.
