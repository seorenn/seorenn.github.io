Emacs 설치

Seorenn Static Pages

Emacs 를 설치하는 방법은 여러가지가 있습니다만 여기서는 macOS 에서 Homebrew 를 통해 설치하는 것을 가정하여 다음의 방법으로 설치할 수 있습니다.

macOS + Homebrew

기본적으로 GNU Emacs 는 아래의 커맨드로 설치가 쉽게 되는 편입니다. 

brew install emacs

다만 최근의 Emacs formulae 에서 --with-cocoa 옵션이 사라진 것으로 유추되어서 이렇게 설치하게 되면 GUI가 없는 터미널 전용 버전으로 설치가 됩니다. 따라서 GUI가 필요하다면 이 방법은 이 글을 쓰는 현재는 추천하지 않습니다.

만약 특별히 빌드할 필요가 없다면 cask 를 이용해 GUI 옵션이 포함된 바이너리를 설치할 수도 있습니다. 

brew cask install emacs

이 방법은 macOS에서 GNU Emacs 를 가장 빠르게 설치할 수 있는 방법입니다.

혹은 GNU Emacs 오리지널이 아니라 약간 튜닝된 버전을 원한다면 Emacs Plus를 설치할 수도 있습니다. 

brew tap d12frosted/emacs-plus
brew install emacs-plus

Emacs Plus 는 기본적으로 스페이스맥스 아이콘으로 표시됩니다. 이것이 싫다면 아래와 같이 빌드 옵션을 줘서 빌드할 수 있습니다. 설치하면 됩니다. 

brew install emacs-plus --without-spacemacs-icon

나머지 OS용 Emacs 설치 방법도 구글에서 검색하면 쉽게 찾을 수 있습니다.
