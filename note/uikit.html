UIKit

2020년 6월 11일 수정

UIKit은 iOS용 UI 프레임워크로 사싱상 아이폰 앱을 만든다면 꼭 필요한 프레임워크 이름이다.

import UIKit

다른 이름으로 Cocoa Touch라고 부르기도 한다. Cocoa(AppKit)를 모바일 터치 인터페이스 용도로 포팅(?)한 것이라 이름이 이렇게 붙은 것 같다.

AppKit과 확연하게 다른 점은 터치 인터페이스 위주로 구성되어 있다는 점도 있겠지만, 각 클래스의 prefix가 UI로 시작된다는 점이다. 예를 들어 뷰 컨트롤러 클래스 이름은 UIViewController 다.
