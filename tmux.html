Tmux 는 터미널 멀티플렉서라고 불리우는 도구입니다. 쉽게 말해서 하나의 터미널을 여러개로 나누어서 쓸 수 있게 만들어줍니다. 현대적인 터미널의 멀티탭이나 멀티Pane 기능과 비슷한 기능을 구현해줍니다.

이런거 왜 쓰냐구요? 글쎄요? 키보드만으로 모든 것을 제어할 수 있다는 것은 편리함을 떠나서 희열이 느껴지진 않나요?😋

비슷하면서도 원조격인 screen 이라는 도구에 영향을 많이 받은 것으로 보입니다.

사용법

그냥 실행시키면 실행은 됩니다. 

tmux

실행된 상태에서 detach, 즉 tmux를 백그라운드 상태로 두려면 아래 단축키 항목에서 백그라운드로 돌리기 항목을 참조하거나, 혹은 tmux 내부의 셸에서 아래 커맨드를 입력하면 됩니다. 

tmux detach

백그라운드에 있는 tmux 세션을 그대로 살리려면 아래 커맨드를 입력하면 됩니다. 

tmux attach

설정법

~/.tmux.conf 파일에 원하는 설정을 기록할 수 있습니다.

리더키(Prefix) 변경하기

기본 리더키는 ^b (Ctrl-B) 입니다. 이걸 바꾸려면, 예를 들어 리더키를 ^\ 로 바꾸려면 아래의 설정을 추가하면 됩니다. 

unbind-key C-b
set-option -g prefix C-'\'

Emacs 등을 쓴느 경우 기본 리더키인 ^b 가 중복되기 때문에 이런 식으로 바꾸는 것을 추천합니다. 물론 필요하거나 원하는 단축키로 설정하면 됩니다.

리더키는 대안키(alternative, prefix2)도 지원됩니다. 기본 대안키는 ^a 입니다. 이것 역시 비슷하게 바꿀 수 있습니다. 

unbind-key C-a
set-option -g prefix2 C-']'

단축키

아래에 언급하는 모든 단축키는 리더키(Prefix)를 먼저 누른 후 눌러야 합니다. 참고로 리더키는 앞서 언급한 대로 사용자가 커스터마이징 할 수 있으며 기본값은 ^B (Ctrl-B) 입니다.

  • c 새 창 만들기(New window)
  • 1 1번 윈도우로 전환(Switch to window 1). 이런 식으로 1에서 9번까지 쓸 수 있습니다.
  • % 창을 좌우로 분할(Horizontal split pane)
  • " 창을 위아래로 분할(Vertical split pane)
  • q 1 창의 나눠진 부분에서 1번 pane으로 이동하기(Switch pane number 1). q 를 누르면 화면에 숫자가 표시되니 원하는 숫자를 누르면 됩니다.
  • o 다른 창으로 포커스 이동
  • h 혹은 j 혹은 k 혹은 r 혹은 커서키: 해당 방향의 pane으로 이동합니다. hjkl 을 아시는 분은 아시겠지만 Vim 의 커서 이동키와 동일한 배치지요.
  • [ 스크롤 모드: 커서키로 스크롤을 할 수 있습니다. 빠져나올 때는 RET (Enter) 키를 누릅니다. 참고로 아래 관련글 항목의 설정을 참고하시면 마우스 휠로도 스크롤은 잘 됩니다.
  • d 백그라운드로 돌리기(detach)

기타 팁

선택영역 복사하기

마우스로 드래그하소 ^C 혹은 Command-C 를 눌렀는데 복사가 안 된다면 ^W (Ctrl+W)를 눌러보세요. 물론 설정에 따라 터미널에서의 복사가 안 될 수도 있습니다.

macOS 유저라면 reattach-to-user-namespace 라는 도구를 설치해야 설정해야 할 수도 있습니다.

종료하지 않고 설정 다시 읽어들이기

.tmux.conf 파일을 수정했는데 지금 열려있는 창들을 닫기가 너무 아깝다면 tmux를 종료하기가 싫을 것입니다. tmux를 종료하지 않고 바꾼 설정을 적용하고 싶다면 어떻게 해야 할까요?

이럴 때는 리더키(Prefix)를 누르고 아래 커맨드를 입력해 봅시다. 

:source-file ~/.tmux.conf

: 키를 누르는 순간 마치 Vim 처럼 하단에 입력 모드가 생기면서 저렇게 입력이 됩니다. 그리고 엔터를 누르면 tmux가 설정 파일을 다시 읽어들입니다.

혹은 다른 방법으로 tmux 안에 아무 셸에서 아래 커맨드를 입력해 봅시다. 

tmux source-file ~/.tmux.conf

이렇게 해도 tmux를 종료하지 않고 설정파일을 다시 읽어들일 수 있습니다.
