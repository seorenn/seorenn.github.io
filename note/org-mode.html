Doom Emacs Snippets

아래는 Doom Emacs에서 Org Mode 용으로 제공되는 스니펫 중 일부다. 아래 키워드를 입력하고 TAB 을 누르면 자동으로 블록 템플릿이 완성된다.

<s
소스 블록(BEGIN_SRC ~ END_SRC)
<q
인용 블록(BEGIN_QUOTE ~ END_QUOTE)
<h
HTML 출력 블록(BEGIN_EXPORT html ~ END_EXPORT)
