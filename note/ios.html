iOS

iOS 는 애플의 모바일 디바이스를 위한 전용 운영체제(Operating System)다. 초기에는 아이폰, 아이패드, 애플 워치 등 다양한 디바이스에 탑재되었다.

완전히 독자적으로 개발된 OS 라기 보다는 macOS를 모바일에 맞게 변경한 것으로 유추된다. 왜냐하면 최초에는 OS X for iPhone 이라는 이름의 OS였기 때문이다.

물론 macOS와는 다른 점이 상당히 많다. 모바일에 맞는 기본 앱 등은 macOS에는 없는 것 들이다. 거기다 요즘은 처지가 역전 되었는지 iOS 용으로 먼저 개발된 소프트웨어가 macOS 용으로 포팅 되는 경우도 종종 보인다.

최근에는 모바일 제품 별 전문화 전략으로 iOS는 아이폰 전용 OS가 되었다. 즉 각 디바이스에 맞게 각각의 OS로 분리되기 시작했다. 예를 들어 아이패드의 경우는 iPadOS라는 별도의 OS가 생겼다.

맥과 아이폰의 점유율 차이로 인해 이제는 부모 격에 해당하는 macOS 보다도 비중이 더 높아졌다고 볼 수 있다. 대부분의 애플 플랫폼 용 앱 개발 튜토리얼은 iOS 를 기준으로 하고 있으며 macOS 용 앱 개발에 관한 자료를 찾기 힘든 점이 이를 반영한다고 생각한다.

릴리즈 내역

iOS 16.1 (2022년 10월)

  • iCloud 공유 사진 보관함
    • 별도의 보관함에서 최대 5명의 다른 사용자와 사진 및 비디오를 매끄럽게 공유할 수 있음
    • 사용자가 보관함을 설정하거나 보관함에 참여할 때 설정 규칙을 사용하여 특정 시작일 또는 사진 속 인물을 기준으로 이전 사진을 추가할 수 있음
    • 공유 보관함 또는 개인 보관함 중 하나만 표시하거나 두 보관함을 함께 표시하고 싶은 경우 보관함 필터를 사용하여 쉽고 빠르게 전환할 수 있음
    • 편집 내용 및 권한을 공유하여 모든 참여자가 사진을 추가, 편집, 즐겨찾기 및 삭제할 수 있음
    • 카메라 앱 내 공유 토글을 사용하여 방금 찍은 사진을 바로 공유 보관함으로 보내거나, Bluetooth를 사용하여 다른 참여자가 근처에 있는 것을 감지했을 때 자동으로 공유하도록 하는 설정을 활성화할 수 있음
  • 실시간 현황
    • iPhone 14 Pro 모델의 경우 Dynamic Island 및 잠금 화면에서 타사 앱의 실시간 현황을 확인할 수 있음
  • 지갑
    • 키 공유 기능으로 지갑 앱에서 메시지 및 WhatsApp 등의 메시지 앱을 통해 차 키, 호텔 객실 키 및 기타 키를 안전하게 공유할 수 있음
  • 홈
    • 스마트 홈의 새로운 연결 표준인 Matter를 지원하여, 다양한 종류의 스마트 홈 액세서리가 생태계 전반에 걸쳐 함께 작동할 수 있도록 함
  • 도서
    • 사용자가 독서를 시작하면 읽기 도구 제어기가 자동으로 가려짐
  • 버그 수정
    • 메시지 앱의 대화 목록에서 삭제된 대화가 계속 나타날 수 있는 문제
    • 화면 상단 간편 접근을 사용할 때 Dynamic Island 콘텐츠를 사용할 수 없는 문제
    • VPN 앱을 사용할 때 CarPlay 연결에 실패할 수 있는 문제
