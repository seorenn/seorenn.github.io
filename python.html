파일 혹은 디렉토리(폴더) 관련

경로에서 파일명만 얻기

>>> import os
>>> os.path.basename('/foo/bar/test.txt')
'test.txt'

파일명에서 확장자 분리하기

>>> import os
>>> os.path.splitext('test.txt')
('test', '.txt')

splitext 는 파일명 뿐만이 아니라 경로(path)에서도 동작합니다. 예를 들어 /foo/bar/text.txt 를 입력할 경우 /foo/bar/text 와 .txt 로 분리됩니다.

파일 혹은 디렉토리 구분하기

>>> import os
>>> os.path.isfile('/foo/bar/some/file.txt')
True
>>> os.path.isdir('/foo/bar/some/file.txt')
False

디렉토리 안의 파일 목록 얻기

>>> import os
>>> os.listdir('.')
['foo.a', 'bar.b', 'somedir', ...]
