pip는 Python 용 패키지를 설치하거나 관리할 때 사용하는 CLI 도구다.

간단한 사용법

패키지 검색

pip search PACKAGE_NAME

패키지 설치

pip install PACKAGE_NAME

특정 버전 패키지 설치

pip install PACKAGE_NAME==VERSION

패키지 설치 및 업그레이드 하기

pip install --upgrade PACKAGE_NAME

패키지 제거

pip uninstall PACKAGE_nAME

패키지 리스트 얻기

pip freeze

위의 명령은 설치된 패키지 목록을 보여 주는데, 보통은 아래처럼 파일 목록에 리스트를 받아둔다. 

pip freeze > requirements.txt

requirements.txt 라는 이름은 패키지 목록을 보관하는 파일로 관례처럼 짓는 이름이다. 혹여나 Python 프로젝트에서 이 파일이 보인다면 이게 의존성 목록이라고 생각하면 된다.

패키지 일괄 설치하기

아래는 앞서 구한 목록인 requirements.txt 파일을 이용해 패키지를 일괄 설치하는 커맨드다. 

pip install -r requirements.txt

특수한 사용법

패키지의 버전 정보 얻기

패키지 버전 리스트를 얻기 위한 별도의 커맨드는 없고, 대신 아래 트릭으로 구할 수 있다. 

pip install PACKAGE_NAME==

특정 버전 설치 커맨드에서 버전을 빼버린 것인데, 이러면 오류가 나면서 설치 가능한 버전을 알려주기 때문에 버전 리스트 구하는 용도로도 사용할 수 있다.
