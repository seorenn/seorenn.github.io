문자열 치환(String Interpolation)

`value = ${value}`

백틱으로 둘러싼다는 점에 유의하자. 중괄호 내부는 변수 뿐만 아니라 식(expression)도 얼마든지 사용할 수 있다.

substring

var str = 'abc123def';
str.substring(3, str.length - 2);
// => 123d

at

Javascript Arrays에서 지원하는 것과 비슷하게 at 메서드는 문자열에서도 사용할 수 있다. 예를 들어 음수를 이용해서 끝에서부터 문자를 읽을 수 있다.

let sample = "This is test"
sample.at(-1)
// => 't'
sample.at(-2)
// => 's'
