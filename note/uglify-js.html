UglifyJS는 Javascript 파서(Parser) 겸 축소기(Minifier) 및 압축기(Compressor)다. 근데 뒤의 두 개는 거의 동일한 의미 같으니 그냥 압축기라고 하자.

이 글은 이 도구의 CLI 사용법을 정리한다.

설치

npm으로 쉽게 설치할 수 있다.

npm i -g uglify-js

사용하기 전에 설치된 버전을 확인해보자.

$ uglifyjs --version
uglify-js 3.9.4

버전이 3 이상이면 ES6 이상에서도 사용이 가능하다.

사용하기

역시나 이 도구도 사용 방법이 많다. 단지 일반적인 압축 상황이라면 아래 커맨드를 자주 사용할 수 있을 것 같다.

uglifyjs -c -m -o jquery.min.js jquery.js

기타 사용 방법은 도움말을 찾아보자.

uglifyjs --help
