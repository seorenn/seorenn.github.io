Lisp 이라는 이름에 어울리게 리스트(list) 처리에 최적화된 프로그래밍 언어입니다. 주로 인공 지능에 많이 쓰인다는데 본 적이 없어서 모르겠네요. 어쨌거나 특수한 용도에서만 종종 사용되는 언어입니다.

Common Lisp

보통 리습(Lisp) 이라는 언어는 일반적으로 Common Lisp 이며 다른 별종(?)들과는 구분 짓는 것이 일반적입니다.

Emacs Lisp

Emacs Lisp 은 Emacs에서 사용되는 리습으로 Common Lisp 과 문법은 동일하지만 기본적으로 제공되는 함수들이 이맥스 에디터 구현을 위해 제공되는 것들이 많습니다. 덕분에 Emacs의 기능을 확장하는데도 유리하게 사용됩니다.

Clojure

리습에서 파생된 언어 중 그나마 활발하게 사용되는 언어가 바로 이 클로져같습니다. 중간언어 역할을 하기 때문에 자체적으로 인터프리팅 되기 보다는 JVM 이나 .Net 바이트코드 등등 다양한 방식으로 컴파일되거나 변환되어 사용됩니다.
