pngquant는 PNG 이미지 포맷을 압축하는 오픈소스 CLI 도구이자 라이브러리다. 압축이라고 표현했지만 사실상 팔레트(palette)를 사용하는 방식으로 색상 수를 최소화 시켜서 이미지를 디더링(dithering) 등을 동원해 최적화 하는 방식으로 파괴 압축(lossy compression)을 하는 것으로 추정한다.

설치

macOS에서는 Homebrew로 쉽게 설치할 수 있다. 

brew install pngquant

이 외에도 🌏공식 사이트에서 다양한 플랫폼 용 설치 방법을 찾을 수 있다.

사용법

단순히 이미지 파일을 인자로 넘기면 압축해 준다. 

pngquant image.png

이렇게 하면 image-fs8.png 라는 파일 이름으로 압축된 파일이 생성된다. fs8 이라는 이름은 압축 방식과 관련된 이름으로 추측된다.

옵션으로 색상 수를 조절할 수도 있다. 

pngquant 64 image.png

이렇게 하면 64색으로 이미지를 최적화 시키기 때문에 색상 수가 많이 필요한 사진 등은 디더링이 생길 수도 있다.

일반적인 *NIX CLI 도구들 처럼 파이프도 사용이 가능하다. 

... | pngquant 32 | ...

파일 이름이 주어지지 않으면 태연하게 stdin으로 입력 받아서 stdout으로 출력해 주는 친절한 도구다.

이 외에도 많은 옵션이 제공되니 --help 로 살펴보자.
