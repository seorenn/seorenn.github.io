grex는 정규표현식 작성을 도와주는 라이브러리 겸 CLI 도구다.

설치

개인적으로는 Homebrew로 설치하는 게 편한 것 같다.

brew install grex

이외에도 Cargo 등으로도 설치가 가능하다. 즉 Rust 라이브러리 패키지도 제공된다.

CLI 사용법

단순하게 매치하려는 문자열을 나열하면 여기에 해당하는 정규표현식을 생성해준다.

$ grex a b c
^[a-c]$
$ grex "this is apple" "this is banana"
^this is (?:banana|apple)$

물론 이 외에도 많은 사용법이 있으니 공식 저장소의 매뉴얼을 참고하자.
