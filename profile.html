Seorenn

네 Seorenn 입니다. 한국어로 '세오렌' 이라고 대충 읽으면 됩니다. 잡다한 일을 하는 개발자이고 고전양식 혹은 르네상스양식 건축물 좋아합니다.

제법 낡은(?) 개발자이기 때문에 터미널로 UNIX 환경에서 Vim 으로 편집하는 것을 즐겼지만, 동시에 Emacs 또한 오랫동안 사용해왔었고, 요근래 한동안은 Spacemacs 를 사용하다 최근에는 Doom Emacs 로 전향한 상태입니다.

좋아하는 프로그래밍 언어는 Swift 와 Python 을 꼽습니다.

들여쓰기는 탭 보다는 스페이스 4칸을 선호합니다.

아 난 도데체 뭘 쓰고 있는 것일까😏
