Doom Emacs의 Org Mode에서 짧은 키로 여러 템플릿을 완성할 수 있는 스니펫(snippets) 즉 단축어 기능을 정리하는 글이다. Org Mode 메이저 모드 상태에서 아래 키워드를 입력한 후 Tab 키를 누르면 템플릿이 자동으로 완성된다.

블록 완성

<s
소스 블록(BEGIN_SRC ~ END_SRC)
src
소스 블록 (위와 동일)
<q
인용 블록(BEGIN_QUOTE ~ END_QUOTE)
quote
인용 블록 (위와 동일)
<h
HTML 출력 블록(BEGIN_EXPORT html ~ END_EXPORT)
elisp
Emacs Lisp 소스 블록(BEGIN_SRC ~ END_SRC 이용) 생성

문법 완성

img
이미지 주소와 제목 템플릿
