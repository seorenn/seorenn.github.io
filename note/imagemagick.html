ImageMagick(이미지 매직)은 다양한 이미지 편집이나 변환 기능을 제공하는 다양한 CLI 도구를 제공하는 패키지다. 이름 끝의 k 는 오타가 아니라 원래 이런 이름이다.

설치

macOS에서는 Homebrew를 이용해 쉽게 설치할 수 있다.

brew install imagemagick

각 리눅스용 배포판에도 맞는 패키지가 제공되므로 고민할 필요 없이 ImageMagick 이름으로 패키지를 검색해보자.

사용법

CLI 도구인 많큼 너무나 많고 다채로우면서도 복잡한 커맨드를 제공한다. 주로 convert 라는 명령을 자주 사용하게 되는데, 일일이 설명하기에는 좀 딱딱할 수 있으니 예제 위주의 ImageMagick 사용법 정리 글을 참고하자.
