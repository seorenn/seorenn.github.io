HTTP2 지원 여부 확인하기

특정 사이트가 HTTP2 를 지원하는지 확인하기 위해서 아래 커맨드를 사용할 수 있다.

curl -I --http2 URL

만약 위 명령의 결과로 해당 사이트가 HTTP2를 지원한다면 아래와 같은 내용으로 시작하는 응답이 온다.

HTTP/2 200

HTTP2를 지원하지 않는다면 지원하는 버전으로 응답이 온다.

HTTP/1.1 200 OK

구분은 가능할 것 같다.
