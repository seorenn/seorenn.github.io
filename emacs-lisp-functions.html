이 글은

그냥 사용해보다 기록해 둘 만한 Emacs Lisp 의 함수들을 정리해보는 글입니다. 아직은 내용이 별로 없지만 함수가 엄청나게 많기 때문에 글도 점점 불어날거라 예상되며 당연하게도 내용이 길어지면 여러 파일로 분리되어 정리될 수도 있습니다.

변수

setq

전역 변수를 생성합니다. 

(setq variable-name 100)
(setq v1 1
      v2 2
      v3 3)

let

지역 변수를 생성합니다. 

(let (a b)
  ...)

초기값도 사용 가능합니다. 

(let ((a 1)
      (b 2)
      (c 3))
  ... body ...)

let*

지역 변수를 생성하는데 앞서 생성한 값을 생성하는데 참고할 수 있는 특수한 형태입니다. 

(let* ((a 1)
       (b (+ a 2))
       (c (/ b 3)))
  ... body ...)

리스트 관련

아이템 꺼내오기

(car '("a" "b" "c"))            ;; "a"
(nth 3 '("a" "b" "c" "d" "e"))  ;; "d"
(car (last '("a" "b" "c")))     ;; "c"

리스트 연장하기

(cons "a" '("b" "c"))           ;; ("a" "b" "c")
(cons '("a" "b") '("c" "d"))    ;; (("a" "b") "c" "d")
(append '("a" "b") '("c" "d"))  ;; ("a" "b" "c" "d")

리스트 추출하기

(cdr '("a" "b" "c"))           ;; ("b" "c")
(nthcdr 2 '("a" "b" "c" "d"))  ;; ("c" "d")
(butlast '("a" "b" "c"))       ;; ("a" "b")

리스트 변수 다루기

(let ((lst '("b" "c")))
  (push "a" lst)
  ;; lst -> ("a" "b" "c")
  (pop lst)  ;; -> "a"
  ;; lst -> ("b" "c")
  )

number-sequence

(number-sequence 3)      ;; (3)
(number-sequence 0 4)    ;; (0 1 2 3 4)
(number-sequence 0 9 2)  ;; (0 2 4 6 8)

mapcar

리스트를 각 아이템을 특정 함수로 평가해서 리스트를 작성합니다. 특정 언어에서는 map 이라는 이름으로 연상할 수 있습니다. 

(defun make-double (x)
  (+ x x))

(mapcar #'make-double '(1 2 3 4 5))
;; -> (2, 4, 6, 8, 10)

mapconcat

리스트를 하나의 문자열로 합칠 때 사용합니다. 

(mapconcat #'identity '("hello" "world" "!") " ")
;; -> "hello world !"

위 예제에서 마지막 매개변수는 각 문자열이 합쳐질 때 사이에 들어가는 문자열입니다.

참고로 위 예제에서 identity 는 입력된 값을 그대로 돌려주는 함수입니다.

문자열 관련

length

(length "abcd")

근데 length 는 다용도라 리스트 갯수 세기도 된답니다.

concat

문자열을 이어붙일 때 사용합니다. 

(concat "hello" " " "world!")
;; -> "hello world!"

substring

(substring "abcde" 2 4)  ;; -> "cd"

string-trim

문자열 좌우의 공백을 제거합니다. 

(string-trim "  blah blah ")
;; -> "blah blah"

format

특정 포맷으로 문자열을 생성합니다. 

(format "I am %d age %s" 19 "old")
;; -> "I am 19 age old"

string-remove-prefix, string-remove-suffix

문자열 앞 혹은 뒤쪽의 특정 문자열을 잘라냅니다. 

(string-remove-prefix "aaa" "aaabbbccc")
;; -> "bbbccc"
(string-remove-suffix "ccc" "aaabbbccc")
;; -> "aaabbb"

string-match

문자열에서 일부분을 비교해서 위치를 리턴합니다. 정규표현식도 사용할 수 있습니다. 

(string-match "bbb" "aaabbbccc")
;; -> 3

number-to-string

(number-to-string 3)

반대로 string-to-number 도 있으니 참고합니다.

조건문

if

(if (condition)
    (then-statement)
  (else-statement))

condition 값은 nil 혹은 () 인 경우에만 False이고 나머지는 True라고 생각하면 됩니다.

then-statement 와 else-statement는 둘 다 함수 하나만 쓸 수 있게 되어있습니다. 만약 여러 프로시져를 넣고 싶다면 progn 으로 블록을 정의해서 써야합니다. 

(if (condition)
    (progn
      (statement 1)
      (statement 2))
  (progn
    (statement 3)
    (statement 4)))

and, or, not 등은 함수로 쓸 수 있습니다. 

(and cond1 cond2 ...)
(or cond1 cond2 ...)
(not (condition))

else 처리가 필요 없다면 when 이나 unless 를 쓸 수도 있습니다. 

(when (condition)
  (statement1)
  (statement2))

(unless (false-condition)
  (statement1)
  (statement2))

when 과 unless 는 progn 을 쓰지 않아도 여러 프로시져를 나열할 수 있습니다.

비교 함수

(equal 10 10)
(equal "str" "str")
(equal '(1 2 3) '(1 2 3))
(equal 'symb 'symb)

타입 확인

(integerp 10)
(floatp 3.14)

string= 혹은 string-equal

문자열이 동일한지 비교합니다. 

(when (string= "aaa" "aaa")
  (message "both are aaa!")

루프

while

(while (condition)
  (statement1)
  (statement2)
  ...)

함수 형식

(defun function-name (param1 param2 ...)
  "documentation"
  ... body ...)

함수를 Emacs 커맨드로 쓰려면 interactive 를 선언해 주어야 합니다. 

(defun function-name (...)
  "documentation"
  (interactive)
  ... body ...)

버퍼 관련

char-after

(char-after 1)

insert

(insert "text")

generate-new-buffer

이름대로 버퍼를 생성합니다. 

(generate-new-buffer "buffer-name")

리턴되는 값은 버퍼의 이름인데 꼭 지정된 이름과 반드시 일치하지는 않습니다.

switch-to-buffer

버퍼를 선택합니다. 

(switch-to-buffer (generate-new-buffer "some-buffer"))

with-current-buffer

현재 선택된 버퍼로 뭔가를 할 때 사용합니다. 

(with-current-buffer some-buffer
  (insert "sample text"))

with-temp-buffer

임시로 빈 버퍼를 만들어 뭔가를 할 때 사용합니다. 파일 내용을 읽어들일 때나 새로운 파일을 작성할 때 등등 의외로 자주 사용됩니다. 

(with-temp-buffer
  (insert "hello world!")
  (write-file "helloworld.txt"))

선택영역 관련

region-beginning, region-end

(goto-char (region-beginning))
(goto-char (regin-end))

커서 관련

goto-char

특정 위치로 커서를 옮깁니다. 

(goto-char (point-max))

위 코드는 현재 버퍼의 제일 끝으로 커서를 옮깁니다.

파일/디렉토리 관련

file-relative-name

특정 경로를 기준으로 원하는 파일 경로의 상대 경로를 구합니다. 

(file-relative-name "/foo/bar/name" "./")
;; -> 당연하게도 사용자 환경에 따라 결과가 다릅니다

file-name-nondirectory

경로에서 파일 이름만을 가져옵니다. 

(file-name-nondirectory "/foo/bar/name.ext")
;; -> "name.ext"

expand-file-name

축약 경로를 절대 경로로 풀어줍니다. 

(expand-file-name "~/test.txt")

file-expand-wildcards

아래 커맨드는 ~/foo/bar 디렉토리 안의 모든 .org 파일 목록을 돌려줍니다. 

(file-expand-wildcards "~/foo/bar/*.org")

file-exists-p

파일이 존재하면 non-nil을 리턴합니다. 

(when (file-exists-p "/foo/bar/file/name")
  (message "file exists!"))

기타

run-at-time

특정 시간 이후에 비동기로 코드를 실행시킬 때 사용할 수 있습니다. 

(run-at-time 1 nil
             (lambda ()
               (message "after 1 seconds...")))
